# Customer Retention & Churn Analysis (Task 2)

Track: Data Science & Analytics (DS)

## Objective
Analyze customer data to identify churn patterns, retention drivers, and customer lifetime trends for a subscription-based business.

## Tools Used
- Power BI
- Excel

## Key Insights
- Customers on monthly contracts have higher churn
- New customers churn more in first 6 months
- Auto-payment users have better retention
- Higher monthly charges lead to increased churn

## Recommendations
- Promote yearly subscription plans
- Improve onboarding for new customers
- Provide loyalty rewards for long-term users
- Encourage auto-payment methods

## Deliverables
- Power BI Dashboard
- Dataset
- Screenshots
- Analysis Report
